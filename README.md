NetScan - Network Vulnerability Scanner

Async Python tool for automated network enumeration, port scanning, service fingerprinting, and banner grabbing with vulnerability analysis.
Features:
- Async network scanning - Discovers live hosts across subnets using asyncio
- Multi-port scanning - Tests 80+ common ports (SSH, HTTP/HTTPS, FTP, SMTP, routers, cameras)
- Service detection - Identifies Apache, nginx, TP-Link, Mikrotik, IoT devices via banner grabbing
- Banner analysis - Grabs HTTP/SMTP/SSH banners, extracts server versions & keywords
- Vulnerability hints - Flags login panels, admin pages, default configs (http/, login, admin, welcome)
- Cross-platform - Linux/Windows/macOS support
